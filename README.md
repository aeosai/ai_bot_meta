Local host ports and related settings need to be configured correctly for everything to function on your computer.

If you decide to use a server later, those server details will need to be configured instead. However, I recommend starting locally and establishing a stable foundation before moving anything to a server.

The same approach applies to the Ravens. While some sections are quite advanced, the setup process remains similar.

I've organised everything by ease of use, prioritising the components that can be worked with and utilised immediately.

The Economy folder may appear large, but all of the contents are interconnected and form part of the same overall system, including the Tier 1, Tier 2, and Tier 3 components of each feature.
